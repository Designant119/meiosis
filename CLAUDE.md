核心强制要求：严格遵循高中及大学基础生物学减数分裂机制，结构、行为、时序绝对无科学性错误；完整还原染色体、着丝粒、姐妹染色单体、同源染色体等全部结构；所有姐妹染色单体差异化配色，视觉清晰区分；动画流程精准、结构建模标准。
一、整体开发硬性规则（必须100%遵守，禁止自主修改）
1. 技术栈限制：仅使用原生 HTML5 + CSS3 + JavaScript，不引入任何第三方框架、插件、外部资源，代码轻量化、可独立运行，双击HTML文件即可在浏览器正常展示动画。
2. 科学性零容错：减数分裂所有阶段的染色体形态、数量、行为、联会、分离、重组规则完全符合权威生物学教材，禁止出现染色体数量错误、分离方向错误、联会错误、着丝粒分裂时序错误等所有科学性问题。
3. 结构完整建模：必须精准建模四大核心结构：同源染色体、姐妹染色单体、着丝粒、纺锤丝，每个结构独立样式、独立标识，不可合并、不可省略。
4. 配色强制规则：每一条染色体的两条姐妹染色单体使用完全不同的对比色，清晰区分；同源染色体使用色系区分（父方、母方染色体色系差异化），所有染色单体视觉上可明确分辨，禁止同色、近色混淆。
5. 流程完整性：完整展示减数分裂全过程：间期 → 减数第一次分裂（前期、中期、后期、末期） → 减数第二次分裂（前期、中期、后期、末期） → 子细胞生成，每个阶段独立展示、可切换、可暂停播放。
二、细胞结构建模精准要求（核心建模标准，绝对不能出错）
1. 染色体与染色单体建模标准
设定生物标准模型：体细胞染色体数2n=4（2对同源染色体，适合可视化展示，符合教学通用模型）。
- 间期（G2期）：染色体完成复制，每条染色体包含2条完全相同的姐妹染色单体，形态为X形结构，结构饱满、比例标准。
- 姐妹染色单体配色规则：单条染色体的左/右两条姐妹染色单体采用高对比配色（例如：一条正红、一条玫红；一条正蓝、一条浅蓝），严格不同色，全程所有分裂阶段保持配色统一，不变色、不混色。
- 同源染色体区分：一对同源染色体整体色系区分（父方染色体红系、母方染色体蓝系），大小、形态完全一致，符合同源染色体定义。
- 染色单体数量规则：严格遵循时序：间期4条染色体→8条染色单体；减Ⅰ全程8条染色单体；减Ⅱ前期、中期4条染色体8条染色单体；减Ⅱ后期着丝粒分裂，变为8条独立染色体，无染色单体。
2. 着丝粒建模标准（重点强制精准）
- 所有染色体的着丝粒为独立可视化结构，位于染色体正中位置，采用高亮白色/黄色圆点展示，体积适中，清晰可见，不与染色单体颜色混淆。
- 着丝粒分裂时序绝对精准：减数第一次分裂全程着丝粒不分裂，仅同源染色体分离；减数第二次分裂后期着丝粒统一分裂，姐妹染色单体彻底分离，成为独立染色体，该时序绝对禁止出错。
- 着丝粒需绑定纺锤丝连接点，动画中纺锤丝牵引点精准对应着丝粒位置，符合真实细胞分裂机制。
3. 辅助结构建模标准
- 纺锤丝：白色细线动态生成，从细胞两极中心体发出，精准连接染色体着丝粒，分裂过程中随染色体移动伸缩、消失。
- 中心体：细胞两极标志性小圆点结构，分裂前期出现，末期消失，位置固定规范。
- 细胞膜/细胞质：浅色透明细胞质区域，圆形细胞轮廓，清晰界定细胞范围，分裂末期细胞膜向内凹陷完成胞质分裂。
三、各分裂阶段动画精准行为要求（零科学错误）
1. 间期
细胞完整，细胞核完整，染色质为细丝状，完成DNA复制和蛋白质合成，为分裂做铺垫，界面标注“染色体复制，姐妹染色单体形成”。
2. 减数第一次分裂前期
染色质高度螺旋化形成染色体；同源染色体两两联会形成四分体；可展示非姐妹染色单体交叉互换效果（可选，教学标准效果）；染色体散乱分布于细胞核内，核膜逐渐消失。
3. 减数第一次分裂中期
四分体整齐排列在细胞赤道板两侧（重点：不是染色体着丝粒排列在赤道板，是同源染色体成对分列赤道板两侧，绝对不能和有丝分裂混淆）；纺锤丝精准连接每一条染色体的着丝粒。
4. 减数第一次分裂后期
同源染色体彼此分离，非同源染色体自由组合；着丝粒保持不分裂，姐妹染色单体不分离；染色体整体向细胞两极移动，动画平滑自然，无错位、无错误分离。
5. 减数第一次分裂末期
细胞中部凹陷，胞质分裂，一个初级性母细胞分裂为两个次级性母细胞；每个子细胞染色体数量减半（2条染色体，4条染色单体），无同源染色体。
6. 减数第二次分裂前期、中期
染色体散乱分布（前期）→ 染色体着丝粒整齐排列在赤道板上（中期）；细胞内无同源染色体，结构清晰。
7. 减数第二次分裂后期
着丝粒瞬间分裂，每条染色体的两条姐妹染色单体彻底分离，成为两条独立的子染色体；子染色体在纺锤丝牵引下向细胞两极移动，该阶段是核心易错点，必须100%精准。
8. 减数第二次分裂末期
胞质再次分裂，两个次级性母细胞分裂为四个生殖细胞；四个子细胞染色体数目减半，遗传物质均分，模型展示最终结果。

2. 文字标注功能：每个分裂阶段顶部显示阶段名称，下方显示该阶段核心生物学知识点，简洁精准。

4. 错误规避：全程杜绝所有常见减数分裂建模错误（如：减Ⅰ着丝粒分裂、减Ⅰ中期染色体赤道板排列、姐妹染色单体同色、染色体数量错乱等）。
